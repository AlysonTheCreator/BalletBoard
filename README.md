
Sistema desktop desenvolvido para auxiliar professores de dança na organização de aulas, alunos, horários e gestão profissional.

---

##  Sobre o projeto

O **BalletBoard** é uma aplicação desktop criada com o objetivo de centralizar e facilitar a gestão do dia a dia de professores de ballet (ou dança em geral).

A aplicação permite organizar informações importantes como:

* Planejamento de aulas
* Controle de alunos
* Horários
* Documentações
* Gestão financeira básica

---

##  Funcionalidades

 Tela de login com validação de usuário
 Planner de aulas
 Cadastro e listagem de alunos
 Organização de horários
 Área de documentações
 Planejamento financeiro
 Interface gráfica amigável

---

##  Tecnologias utilizadas

* **Java** (JDK 11+)
* **Java Swing** (interface gráfica)
* **SQLite** (banco de dados local)
* **JDBC** (conexão com banco)
* **IntelliJ IDEA** (IDE)

---

##  Interface

O sistema possui uma interface desktop com:

* Tela de login personalizada
* Botões com design moderno
* Navegação simples entre telas

---



##  Banco de dados

O sistema utiliza **SQLite**, permitindo:

* Execução local
* Fácil configuração
* Sem necessidade de servidor

##  Objetivo do projeto

* Prática de Java Desktop
